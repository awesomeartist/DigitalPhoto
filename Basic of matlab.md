### 一、Matlab软件特性

1. 新建脚本和新建实时脚本区别
	- 前者需要保存才可以运行，后者不需要保存即可运行
2. ans
	- ans时默认变量，用于存储当前指令运行后的结果
3. Matlab软件赋值格式
	- 变量=表达式
	- 表达式可以由函数，变量构成
	- 若在表达式后面输入‘；’则不显示表达式运行结果，但其结果任保存在Matlab软件内存中
	- 若一行太长可以用续行符号...将其余部分延续到下一行
	- 一行可以写多个语句，它们之间用逗号或者分号隔开
4. Matlab的注释符号时‘%’
5. Matlab变量命名
	- 第一个字符必须是英文字母
	- 区分大小写
	- Matlab中变量不需要进行事先说明，他会根据变量的值和操作来自动确定变量的类型
	- 若变量存在 会用新的变量类型代替旧的变量类型



### 二、命令行窗口中常用的命令
|    clc     | 清除命令行窗口的显示内容 (变量和值依旧存在)                  |
| :--------: | ------------------------------------------------------------ |
|   clear    | 清楚Matlab软件工作内存中的变量                               |
|    who     | 列出Matlab软件工作内存中驻留的变量名清单                     |
|    whos    | 列出变量名清单和属性                                         |
|    edit    | 打开M文件编辑器                                              |
|   format   | 定义输出格式                                                 |
|   short    | 小数点后4位有效数字                                          |
|    long    | 小数点后15位有效数字                                         |
|  short e   | 用5位有效数字的科学计数法表示                                |
|   long e   | 用15位有效数字的科学计数法表示                               |
|  funtool   | 打开对函数进行简单操作的可视化交互界面，执行后显示如下三个可操作图形窗口 |
| Taylortool | 打开可视化函数图形器，用于观察不同次数Taylor多项式逼近函数本身的状态 |
